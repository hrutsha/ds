
```
import numpy as np
import pandas as pd
```
data
```
data = {
    'A':[1,2,np.nan,4,5],
    'B':[1,2,3,4,5],
    'C':[1,2,3,np.nan,np.nan],
    'D':[1,np.nan,np.nan,np.nan,5],
}
df = pd.DataFrame(data)
```
Finding Missing Data
```
df.isna()
df.isna().sum()
df.isna().any()
```
removing missing data
```
df.dropna()
df.dropna(thresh=3)
```
filling missing data
```
df.fillna(0)
```
```
values={
    'A':0,
    'B':100,
    'C':200,
    'D':400,
}
df.fillna(value=values)
```
```
df.fillna(df.mean())
```
