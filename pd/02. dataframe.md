```
import pandas as pd
```

```
data = {
    'name':['HR','Utsha','Rashid'],
    'age':[25,21,26],
    'city':['Dinajpur','Bogura','Khulna'],
    'salary':[20000,21000,22000]
}
```

```
df = pd.DataFrame(data)
```
comumn selection
```
df['name']
df[['name','age']]
```
rows selection
```
df.loc[0]
df.loc[[0,1]]
```
selecting subsets of rows and columns
```
df.loc[[0,1]][['city','salary']]
```
conditional selection
```
df[df['age']>25]
df[(df['age']>20) & (df['city'] == 'Dinajpur')]
```
delete column
```
df.drop('designation',axis=1)
df.drop(['city','salary'],axis=1)
```
